# Nexus Red Team Portal

Static final term project prototype for BIS 23 Client-Side Programming.

Pages included:
- Home dashboard
- About / business page
- Applications portfolio page
- Contact page
- The Arsenal search page
- Breach Report prototype page
- Phish-Lab prototype page
- Admin Logs review page

Integrated applications:
- Miles Per Gallon Calculator
- Auto Image Slider with controls and autoplay
- Security Readiness Assessment

Deployment:
- Upload the contents of this folder to a GitHub Pages repository.
- Use `index.html` as the site entry point.
- All pages are static HTML, CSS, and JavaScript with no build step required.

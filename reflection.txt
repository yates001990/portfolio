Short Reflection

The two applications included in Nexus Red Team Portal are the Miles Per Gallon Calculator and the Auto Image Slider. The MPG Calculator accepts mileage and gallons-used values, computes fuel efficiency, and labels the result as Efficient, Average, or Fuel Heavy. The Auto Image Slider rotates through four personal images with previous/next controls, autoplay on a timed interval, a slide counter, and clickable thumbnail navigation. Both applications are fully functional JavaScript demos that satisfy the course assignment requirements.

Rather than linking to standalone HTML files, each application is embedded inside the portal's shared design system. They use the same header, navigation, footer, fonts, color palette, and CSS layout as every other page on the site. The Applications page introduces each app with a description and a button that routes to its dedicated demo page, all within the site's navigation structure. The biggest challenge was making everything feel like one cohesive product instead of separate classroom exercises. Matching component styles, maintaining consistent spacing, and ensuring the responsive layout worked across all pages including the operational modules like Arsenal, Breach Report, Phish-Lab, and Admin Logs required careful coordination between the HTML structure, the shared stylesheet, and the single JavaScript file that powers all interactivity.
